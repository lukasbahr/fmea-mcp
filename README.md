# FMEA MCP

Python MCP server using Pydantic AI and FastAPI with vector embeddings support for FMEA (Failure Mode and Effects Analysis).

## Description

This project provides a Model Context Protocol (MCP) server for managing and analyzing FMEA data using Neo4j graph database and AI-powered capabilities.

## Installation

```bash
uv pip install -e .
```

## Usage

Run the MCP server:

```bash
python -m fmea_mcp.server
```

## Development

Install development dependencies:

```bash
uv pip install -e ".[dev]"
```
