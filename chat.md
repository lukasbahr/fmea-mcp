Analysis & Calculation Tools
calculate_rpn - Calculate/recalculate Risk Priority Numbers for specific failure modes
rank_failure_modes - Rank failure modes by RPN, severity, or custom criteria
identify_critical_failures - Find failures above RPN threshold or severity level
compare_rpn_changes - Track RPN improvements over time after implementing measures
Query & Discovery Tools
find_common_causes - Identify failure causes that affect multiple process steps
find_similar_failure_modes - Find similar failure modes across different processes (beyond just embeddings)
get_failure_mode_details - Get complete information about a specific failure mode including all relationships
search_by_process_step - Retrieve all failure modes for specific process step(s)
search_by_severity_range - Find failures within S/O/D rangesRelationship Analysis Tools
trace_failure_impact - Follow the chain from cause → failure mode → effect
find_mitigation_gaps - Identify failure modes lacking prevention or detection measures
get_related_measures - Find all prevention/detection measures for a failure cause
analyze_measure_effectiveness - Evaluate which measures reduce RPN most effectively
Pattern Recognition Tools
detect_failure_patterns - Identify recurring patterns (e.g., same cause across steps)
analyze_process_vulnerabilities - Identify which process steps have highest risk concentration
find_common_detection_methods - Group similar detection approaches
Reporting & Export Tools
generate_fmea_report - Create formatted FMEA report for specific process/failure
export_critical_items - Export high-priority items for action planning
get_action_priority_matrix - Generate prioritized action items based on RPN
Data Management Tools
update_failure_mode - Update S/O/D ratings or descriptions

link_corrective_actions - Associate implemented fixes with failure modes
archive_resolved_failures - Mark failure modes as resolved/mitigated
Advanced Analytics Tools
predict_failure_likelihood - AI-assisted prediction based on similar historical failures

Visualization Support Tools
